Giới thiệu về Bootstrap

Bootstrap là gì?

Bootstrap là một framework CSS mã nguồn mở phổ biến nhất trên thế giới, được phát triển bởi Twitter. Nó cung cấp các class CSS và JavaScript để xây dựng giao diện web responsive và hiện đại một cách nhanh chóng.

Tính năng nổi bật:
  • Responsive Design - Tương thích mọi thiết bị
  • Mobile-first approach - Thiết kế ưu tiên di động
  • Pre-built components - Các thành phần giao diện có sẵn
  • Customizable - Có thể tùy chỉnh theo nhu cầu
  • Cross-browser compatibility - Tương thích trình duyệt

Cách nhúng Bootstrap vào Project

Cách 1: Sử dụng CDN (Content Delivery Network)

Đây là cách nhanh nhất để sử dụng Bootstrap mà không cần tải xuống:

https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css

Thêm JavaScript:

https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js

Cách 2: Sử dụng NPM

Nếu bạn sử dụng Node.js, có thể cài đặt Bootstrap qua npm:

npm install bootstrap

Sau đó import vào project:

import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';

Cách 3: Tải xuống trực tiếp

Tải file CSS và JS từ getbootstrap.com

Công dụng của thư viện JavaScript Bootstrap

1. JavaScript APIs

Bootstrap cung cấp API để quản lý các component một cách linh hoạt:

  • Modal: var myModal = new bootstrap.Modal(document.getElementById('myModal'))
  • Toast: var toastEl = document.getElementById('myToast'); var toast = new bootstrap.Toast(toastEl)
  • Carousel: var carousel = new bootstrap.Carousel('#myCarousel')
  • Tooltip: var tooltip = new bootstrap.Tooltip(element)
  • ScrollSpy: Tự động cập nhật navigation khi scroll

2. Event Handling

Xử lý sự kiện với Bootstrap:

element.addEventListener('shown.bs.modal', function () { ... })

3. Tùy chỉnh Options

Có thể tùy chỉnh các component với options:

new bootstrap.Modal(element, { keyboard: false, backdrop: 'static' })

Các vấn đề với !important trong Bootstrap

Cảnh báo: Sử dụng !important

Trong Bootstrap, một số class sử dụng !important để ghi đè CSS. Điều này có thể gây ra các vấn đề khi bạn cố gắng tùy chỉnh giao diện.

Ví dụ về vấn đề:
/* CSS tùy chỉnh của bạn - KHÔNG hoạt động */
.custom-color {
    color: red !important; /* Phải thêm !important để ghi đè */
}
Giải pháp khuyến nghị:
  1. Sử dụng Sass/CSS tùy chỉnh - Định nghĩa biến trước khi import Bootstrap
  2. Tránh ghi đè trực tiếp - Sử dụng cấu trúc HTML phù hợp với Bootstrap
  3. Sử dụng class bổ sung - Thêm class riêng thay vì ghi đè
  4. Tùy chỉnh Bootstrap - Biên dịch Bootstrap từ source với các thay đổi
Ví dụ giải pháp đúng:
/* Cách tốt hơn: Sử dụng custom class */
.my-custom-text {
    /* Không cần !important nếu đặt sau Bootstrap CSS */
    color: #ff0000;
}
Lưu ý: Lạm dụng !important sẽ làm khó bảo trì code và gây ra "CSS specificity wars" - nơi bạn phải thêm ngày càng nhiều !important để ghi đè.
